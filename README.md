# 🌳 Weather Prediction Using Decision Tree Algorithm

A Machine Learning project that predicts whether a person can play outside based on weather conditions using the **Decision Tree Classifier**. This project includes a beautiful **light pink Streamlit UI**, interactive predictions, model evaluation, and decision tree visualization.

---

# 📌 Project Overview

This project uses weather-related features such as:

* Outlook
* Temperature
* Humidity
* Wind

to predict:

✅ **Play Outside**
❌ **Do Not Play Outside**

The model is trained using the **Decision Tree Algorithm** from Scikit-learn.

---

The main goal of this project is to:

* Understand how Decision Trees work

# 🌳 Decision Tree Algorithm

Decision Trees are supervised machine learning algorithms used for:

* Classification
* Regression

They split data into branches based on feature conditions to make predictions.
